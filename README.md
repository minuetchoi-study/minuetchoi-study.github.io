# 베트남어 남부 발음

- giờ -> 여
- có thể -> ó thể
- đặt -> 닥
- vừa -> 여으어, <mark style="background-color: #ffdce0">내가 듣기론 '리어'와 '지어' 사이?</mark>
- rồi -> 로이
- không -> 홍

# 조사

- đấy
  - 딱히 의미는 없다.
  - 의문사(어떤, 어디, 무엇) 뒤에 같이 사용한다.
  - 습관이다.
  - 사용하면 부드럽고 친근감을 느낄 수 있다.
